# Atelier — Stock & Flux (interface interne)

Cette page est volontairement **non indexée** (meta robots) et **non liée** depuis le site.

URL (attention au `%` qui devient `%25` dans le navigateur) :
- `/adch%25stock&flux/`

Principes :
- Stock & prix dérivés d'un **journal de mouvements**.
- Annulation = **contre-écriture** (on n'édite pas l'historique).
- Sauvegarde locale (localStorage) + export/import JSON.

⚠️ GitHub Pages n'est pas un espace privé : ne stocke pas de données sensibles (clients, etc.).
